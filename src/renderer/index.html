폰 연결하기
Android 버전에 맞는 방법을 선택하세요
  1. 설정 → 개발자 옵션 ON 빌드 번호를 7회 탭하면 개발자 옵션이 나타납니다
  2. 무선 디버깅 ON 개발자 옵션 → 무선 디버깅 활성화
  3. 아래 주변 기기에서 검색 클릭 자동으로 디바이스를 발견합니다
  4. 연결 → 미러링 버튼 클릭 연결 즉시 미러링 + AI 제어 모두 사용 가능
  1. 설정 → 개발자 옵션 ON 빌드 번호를 7회 탭하면 개발자 옵션이 나타납니다
  2. USB 디버깅 ON 개발자 옵션 → USB 디버깅 활성화
  3. USB 케이블로 PC와 폰 연결 폰에서 "USB 디버깅 허용" 팝업이 나오면 허용
  4. 무선 전환 버튼 클릭 → USB 분리 이후 무선으로 자동 연결됩니다
